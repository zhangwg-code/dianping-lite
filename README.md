# 点评Lite（简化版大众点评原型）

一个可运行的“简化版大众点评”全栈原型：用户可搜索/浏览商家并发布评价；商家可创建/编辑自己的商家并回复评价。

## 快速开始

1. 安装依赖：`pnpm install`
2. 准备环境变量：复制 `.env.example` 为 `.env`，填入 Supabase 项目配置
3. 启动开发：`pnpm run dev`
4. 访问：`http://localhost:5173/`
